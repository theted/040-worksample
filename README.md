# 040 worksample
Sample application


## Public url
Is available at https://040.herp.se/


## Local setup
Should be easy; database config is stored in `config.php`, database structure in `./data/db.sql`.


## Building static assets

Install dependencies
```
npm install
```

Build static assets
```
npm run build
```

Watch assets; re-build as needed
```
npm run watch
```
